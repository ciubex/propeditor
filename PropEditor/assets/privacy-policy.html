Privacy Policy

ROOT access

The application need ROOT access in order to change device properties.

  1. android.permission.ACCESS_SUPERUSER

Write and read to storage

The application allow the used to write existing device properties to external storage, for this write/read storage permissions are required:

  1. android.permission.WRITE_EXTERNAL_STORAGE

Non Disclosure

The PropEditor does not handle personnal or sensitive user data, and no personnal data is captured or exchanged outside the application.
